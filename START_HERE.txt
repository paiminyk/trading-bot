## 🎯 SIGNAL LOGIC (Simple Explanation)

Bot mencari 2 mode:

### SIDEWAYS Mode
Bot tunggu price ekstrem:
- **BUY** jika price bounces dari bawah
- **SELL** jika price bounces dari atas

### TRENDING Mode
Bot follow trend:
- **BUY** jika price breakout ke atas
- **SELL** jika price breakdown ke bawah

### Score System (0-100)
Sebelum entry, bot cek 4 things:
1. Price alignment dengan MA? (+25 points)
2. Band width wide enough? (+25 points)
3. Volume tinggi? (+25 points)
4. RSI confirm signal? (+25 points)

**Min score 60 untuk entry** ✅

## 📊 REALISTIC EXPECTATIONS

### Signal Frequency
- Expect 1-5 signals per hari (depending pair)
- Beberapa hari bisa no signal (normal)
- Not high-frequency scalping

### Win Rate
- Good target: 55-65%
- Excellent: 70%+
- Depends pair + timeframe + tuning

### Profitability
```
IF:
- 10 trades/month
- 60% win rate (6 wins, 4 losses)
- Avg win: $5
- Avg loss: $5
- Fee: 0.04% per round trip

THEN:
- Gross: (6×$5) - (4×$5) = $10
- Fee: ~$0.10
- NET: ~$10/month

With $25 size, expect $10-50/month
(Tergantung pair volatility & config)
```

⚠️ **Past ≠ Future! Trade with caution!**

## ⚙️ CUSTOMIZATION OPTIONS

Bot bisa customize untuk strategy Anda:

```python
# Ganti pair
SYMBOL = "BTCUSDT"  # dari "CLUSDT"

# Ganti timeframe
TIMEFRAME = "1h"    # dari "15m"

# Ganti size
USDT_SIZE = 50      # dari 25 (bigger risk)

# Ganti TP/SL
TP_PCT = 0.03       # dari 0.02 (+3%)
SL_PCT = 0.01       # dari 0.015 (-1%)

# Adjust signal sensitivity
MIN_SCORE = 70      # dari 60 (stricter)
```

## 🛡️ SECURITY NOTES

✅ **API Key Protection:**
- Tidak ada API key di code
- Semua di GitHub Secrets (encrypted)
- `.gitignore` prevent accidental commit

✅ **Best Practices:**
- Restrict Binance API to IP (recommended)
- Disable withdrawal di API
- Enable 2FA di Binance

⚠️ **Jangan:**
- Share API keys (akan di-hack)
- Push .env files (use secrets instead)
- Use same key untuk multiple bots
